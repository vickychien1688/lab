# PAS English Lab · 英語跟讀系統

一套給英語老師的「跟讀（shadowing）錄音」教學系統。學生邊聽老師示範音檔邊朗讀，
系統把「老師示範 + 學生聲音」混音錄下並上傳給老師；老師有一個密碼保護的後台可以
管理課程、收聽錄音、打分數、看繳交統計。

**零成本、零伺服器**：前端放 GitHub Pages，後端用 Google Apps Script + Google Sheet + Drive。

## 功能

**學生端**（`index.html` → `student.html`）
- 依班級動態載入課文與老師示範音檔（內容由後台管理，不寫死在程式裡）
- 一鍵錄音：播放老師示範的同時錄下自己的朗讀，即時波形顯示
- 試聽後上傳給老師

**老師後台**（`admin.html`，密碼登入）
- 🎧 **學生錄音**：依班級/課次/姓名篩選，線上播放、下載、打分數與評語、刪除
- 👤 **學生繳交**：依學生彙整誰交了幾次、交了哪些課
- 📊 **統計**：總繳交數、學生人數、已/待評分、各課平均分
- 📚 **課程管理**：新增/編輯 班級、課文、示範音檔網址
- ⚙️ **設定**：修改後台密碼

## 檔案結構

```
index.html          學生首頁（動態列出班級）
student.html        學生錄音頁（?class=G7）
admin.html          老師後台
assets/
  config.js         ★ 唯一要設定的地方：填入後端 API_URL
  app.css           共用樣式（延續原本深色主題）
  student.js        錄音／上傳邏輯
  admin.js          後台邏輯
gas/
  Code.gs           後端（貼進 Google Apps Script）
G7/ G8/             示範音檔 mp3
SETUP.md            ★ 部署步驟，請照著做
```

## 快速開始

看 **[SETUP.md](SETUP.md)**：建立 Google Apps Script 後端 → 把 `/exec` 網址填進
`assets/config.js` → 推到 GitHub Pages → 用預設密碼 `1234` 登入後台並立即改密碼。

## 相較舊版的改動

- 課程內容從「寫死在 HTML」改為「後台/試算表管理」，老師自己就能加課、換音檔。
- 上傳改為可讀取後端回應，送出成功/失敗有明確提示（舊版是 no-cors 盲送）。
- 新增完整的老師後台（原本完全沒有）。
- 錄音送出時一併記錄班級、課次、時長，方便後台統計與評分。
