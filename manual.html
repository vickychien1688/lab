1這個平台怎麼運作

系統分三塊，用「派作業」把內容和學生連起來：

  • 📚 資源庫：所有書與課文（例《The Elephant Man》Ch1–4），是你的教材倉庫。
  • 🏫 班級 + 👥 學生：一個班級是一群學生，每班有一組班級代碼給學生登入。
  • 📌 派作業：從資源庫挑一課，指派給某個班級。
老師：整理資源庫 → 建班級（拿代碼）→ 建學生名單 → 派作業
學生：用代碼登入 → 看到自己班的作業 → 錄音上傳
老師：聽錄音 → 打分數＋評語 → 學生登入就看到自己的分數與評語

全部免費、免安裝：學生用手機/電腦瀏覽器就能用。

2👩‍🎓 給學生用（可印給學生或貼在群組）

  1. 用手機 Safari 或 Chrome 打開：read.our-reading-space.com
  2. 輸入老師給的 班級代碼 → 按「進入」。
  3. 點選自己的名字（如果老師有設 PIN，再輸入 4 位數 PIN）。
  4. 看到「📋 我的作業」，點要錄的作業旁的 🎤 去錄音。
  5. 按 🔴 START 跟著唸 → ⏹ STOP → 📤 SEND 送出。
  6. 老師改完後，回「我的作業」就能看到自己的分數和評語。
🔴 一定要用 Safari／Chrome 開，不要直接在 LINE 裡開（LINE 內建瀏覽器不能錄音）。第一次錄音會問麥克風權限，請按「允許」。

3老師：登入後台

  1. 開老師後台網址，輸入密碼（預設 1234）。
  2. 上方分頁：🏫 班級👥 學生名單📌 派作業🎧 學生錄音👤 繳交彙整📊 統計📚 資源庫⚙️ 設定
第一次登入請到 ⚙️ 設定 把密碼從 1234 改掉。（兩位老師共用同一組密碼）

4老師：建班級 & 學生名單

🏫 班級

  1. 按 ＋ 新增班級，填班級名稱（例：週三晚班）和一組班級代碼（英數字，例 abc123）。
  2. 把這個班級代碼發給該班學生，他們就能登入。

👥 學生名單

  1. 在上方選一個班級。
  2. 按 ＋ 新增學生 一個個加；或按 ＋ 批次貼上名單，一行一個名字一次貼上。
  3. 需要的話可幫學生設 PIN 碼（避免同學互點名字）；留空就不用 PIN。
分數是綁在「學生」身上的，所以要先把學生建進名單，之後才對得起來、學生也才看得到自己的成績。

5老師：整理資源庫（📚 書＋章節）

資源庫就是你的書櫃：一本書底下放它的章節（Ch1、Ch2…）。書不綁年級——要派給哪個班，是「📌 派作業」決定的，同一本書可以派給任何班級。

新增書本

  1. 在「書本」按 ＋ 新增書本，填書名（例 Wonder）→ 儲存。就這樣。

幫書加章節（課文＋示範音檔）

  1. 在「課文 / 示範音檔」按 ＋ 新增課文，選書本、填課次ID（例 ch1）、標題、貼課文。
  2. 示範音檔：直接「選擇檔案」上傳電腦裡的 mp3／m4a（推薦），或貼網址。
  3. 按儲存。同一本書要加 Ch2、Ch3 就重複這步，課次ID 換成 ch2、ch3。
⚠️ 同一本書裡的課次ID不可重複；用到已存在的會跳確認提醒（那會覆蓋舊的那課）。

6⭐ 讓學生「一句一句跟讀」的兩種做法

方法一（最省事）· 音檔本身已剪好停頓

音檔已含空白（唸一句→留空白→下一句）：上傳音檔、不要勾「逐句交錯跟讀」、儲存即可。學生連續播放就會在空白處跟著唸。

方法二 · 連續音檔讓系統插空白

  1. 編輯課文，勾 🎧 開啟逐句交錯跟讀。
  2. 按 ▶ 播放，每聽完一句就按「✂ 在這裡分句」（停頓位置由你決定）。
  3. 設空白倍數（空白＝該句長度×倍數；程度弱的班級調高）。儲存。

7老師：派作業（📌）

  1. 到 📌 派作業，上方選要派給哪個班級。
  2. 按 ＋ 指派作業 → 選書本 → 選課次 → （可選）填截止日 → 儲存。
  3. 該班學生登入後，「我的作業」就會出現這一課。
學生只看得到「有派給他班級」的作業。想讓學生練別課，就再派一份。

8老師：聽錄音、打分數＋評語（🎧）

  1. 到 🎧 學生錄音，可用班級/課次/姓名篩選。
  2. 按 ▶ 播放 線上聽（旁邊有 ⬇ 下載）。
  3. 按 ✍ 評分 填分數（0–100）與評語，儲存。
  4. 學生下次登入「我的作業」就會看到 ✅ 已評分：X 分 和老師評語。

10常見問題

🔴 學生用 LINE 點開卻不能錄音（最常見）

LINE 內建瀏覽器（尤其 iPhone）不支援錄音。請學生改用 Safari／Chrome：iPhone 點右下角分享圖示→「用 Safari 開啟」；Android 點右上「⋮」→「用其他瀏覽器開啟」。頁面偵測到 LINE 時也會自動跳橘色提示。

學生登入後說「沒有作業」

代表老師還沒派作業給他的班級，或作業被設成隱藏。到 📌 派作業 指派一份即可。

學生點不到自己名字 / 進不去

確認（1）班級代碼正確、（2）老師已把該學生加進那個班的名單、（3）若有設 PIN，PIN 要對。

想換一個學生登入

在「我的作業」右上按 登出，就會回到登入頁重新輸入代碼。
