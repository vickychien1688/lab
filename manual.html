1這個系統在做什麼

學生打開網頁，一邊聽老師的示範音檔、一邊朗讀，系統會把「老師示範＋學生的聲音」混音錄下來並上傳給你。你在後台可以聽每個學生的錄音、打分數、看誰交了、管理課文與音檔。

全部免費、免安裝：學生用手機或電腦瀏覽器就能用；你用有網路的裝置登入後台即可。

2學生怎麼用（可印給學生看）

  1. 打開學生網址，選自己的班級。
  2. 上方選課次，輸入自己的名字。
  3. 按 🔴 START → 聽到老師示範就跟著唸；一般模式會連續播放，逐句模式會「播一句→留空白換你唸→下一句」。
  4. 結束按 ⏹ STOP，可先試聽。
  5. 沒問題按 📤 SEND TO TEACHER 送出，看到「✅ 已送出」就完成。
第一次會跳出「要求使用麥克風」，請按允許，否則錄不到聲音。建議用耳機，混音效果最好。

3老師登入後台

  1. 打開老師後台網址。
  2. 輸入密碼（預設 1234）按登入。
  3. 進去後有五個分頁：🎧 學生錄音👤 學生繳交📊 統計📚 課程管理⚙️ 設定
第一次登入請到 ⚙️ 設定 把密碼從 1234 改掉。

4管理課程（📚 課程管理）

新增班級

  1. 按 ＋ 新增班級。
  2. 填班級ID（例 G9，建立後別改）、班級名稱、書名/主題，按儲存。

新增 / 編輯課文

  1. 按 ＋ 新增課文（或既有課文按編輯）。
  2. 選班級、填課次ID（例 ch1）、顯示標題、貼上課文。
  3. 設定示範音檔（見下一段）。
  4. 顯示給學生選「是」，按儲存。

示範音檔：兩種方式（擇一）

方式 A（推薦）· 直接上傳
在「示範音檔」按 選擇檔案，挑你電腦硬碟裡的 mp3／m4a，看到「✅ 已上傳」後，記得按最下面儲存才生效。
方式 B · 貼網址
如果音檔已經有網路連結，也可以直接貼在下面的網址欄。
改課文、換音檔都在這裡完成，不需要改任何程式碼。

5⭐ 設定「逐句交錯跟讀」

想讓學生一句一句跟讀（播一句 → 留空白錄音 → 下一句），在編輯課文時：

  1. 先設好示範音檔（上傳或貼網址）。
  2. 勾選 🎧 開啟逐句交錯跟讀。
  3. 按 ▶ 播放 聽音檔，每聽完一句就按一次「✂ 在這裡分句」 —— 停頓位置完全由你決定，點在哪就斷在哪。
  4. 標好的分句點會列成 1. 0:032. 0:08…，可單獨 ✕ 刪或全部清除。
  5. 設空白倍數：空白時間 = 該句長度 × 倍數。程度較弱的班級調高（例 2.0），程度好的調低（例 1.2）。
  6. 按儲存。
同一課文可存不同版本給不同班級（例如 G7A、G7B 各建一課，倍數不同）；不勾選就是原本的連續播放。

6聽錄音、打分數（🎧 學生錄音）

  1. 可用班級 / 課次 / 姓名篩選。
  2. 按 ▶ 播放 線上聽學生錄音；播放器旁有 ⬇ 下載。
  3. 按 ✍ 評分 填分數（0–100）與評語，儲存後狀態變「已評」。
  4. 不要的可按 🗑 刪除（含雲端檔案）。

8改後台密碼（⚙️ 設定）

  • ⚙️ 設定 → 輸入新密碼兩次 → 更新。
  • 兩位老師共用同一組密碼；改了之後記得互相告知。

9常見問題

學生說錄不到聲音 / 沒有老師的示範

  • 確認學生按了麥克風「允許」。
  • 用方式 A 上傳或把 mp3 放在系統同網域最穩；貼外部網址若不支援跨網域，老師示範會錄不進去。

逐句跟讀停頓怪怪的

回編輯課文，重新播放、重點分句點；或調整空白倍數。分句點位置由你決定，可隨時改。

送出失敗

多半是網路或麥克風權限問題，請學生重新整理、確認允許麥克風後再試。
