老師後台 → 完整手冊 →

👩‍🏫 老師：派一份作業（5 步）

1
登入後台
read.our-reading-space.com/admin.html
輸入自己的帳號＋密碼。
2
📚 資源庫：建書＋章節
＋新增書本 → 填書名。＋新增課文 → 選書、填章節標題（CH1）、貼課文、選擇檔案上傳音檔 → 儲存。
3
🏫 班級：建班級拿代碼
＋新增班級 → 填名稱＋自訂一組班級代碼（例 G7-2026）。代碼發給學生。
4
👥 學生名單：加學生
選班級 → 「批次貼上名單」一行一個名字，一次貼完。
5
📌 派作業
選班級 → ＋指派作業 → 選書、選課次 → 儲存。完成！

✍️ 老師：批改

1
🎧 學生錄音 → 按 ▶ 播放 聽
2
按 ✍ 評分 → 填分數＋評語 → 儲存
學生登入就會看到自己的分數和評語。

👩‍🎓 學生：交作業（3 步）

1
用 Safari 或 Chrome 打開
read.our-reading-space.com
2
輸入班級代碼 → 點自己的名字登入
3
點作業的 🎤 去錄音 → 🔴 START 跟著唸 → ⏹ STOP → 📤 SEND
老師改完後，回「我的作業」看分數與評語。
🔴 不要在 LINE 裡直接開（不能錄音）！iPhone：右下分享 →「用 Safari 開啟」；Android：右上「⋮」→「用其他瀏覽器開啟」。第一次錄音請按「允許」麥克風。
